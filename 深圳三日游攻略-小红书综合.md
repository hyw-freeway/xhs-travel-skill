## ✈️ 二、交通与住宿指南

### 大交通（到达/离开）

| 方式 | 说明 |
|------|------|
| **飞机** | 深圳宝安国际机场 → 地铁 11 号线直达福田，约 35 分钟 |
| **高铁/火车** | 优先选 **深圳北站** 或 **福田站**（在市中心，转地铁方便） |

### 市内交通

- **省钱方式**：手机 NFC 绑定「深圳通」或办 **深圳通实体卡** — 地铁约 95 折、公交 8 折。
- **地铁**：市内主要景点地铁可达，可参考「深圳市地铁出行攻略」类笔记查看线路图。
- **去大鹏半岛**：深圳北/市区地铁 → 坪山/大鹏方向公交或旅游专线；自驾更省时。**周末及节假日自驾进大鹏需在公众号预约通行。**

## 📅 三、每日详细行程

### Day 1｜城市地标＋湾区黄昏夜景

| 时段 | 安排 |
|------|------|
| **上午** | 莲花山公园＋市民中心/平安大厦外观 → 可顺路 **深圳博物馆**（室内凉快＋城市历史） |
| **中午** | 早茶：**蘩楼、凤凰楼、点都德**；正餐：**京味张·北京烤鸭**（多篇笔记推荐的地标美食） |
| **下午** | 万象天地/深业上城 Citywalk，或 **深圳湾文化广场**（新地标，展览＋建筑，建议预留 2–3 小时） |
| **傍晚～晚上** | **深圳湾公园**（草地、骑行、看日落）→ **欢乐港湾「湾区之光」摩天轮** 或 **海上世界＋明华轮**（夜景＋晚餐） |

### Day 2｜大鹏半岛海岸线（桔钓沙＋杨梅坑＋大鹏所城）

> ⚠️ 建议尽量避开周末/节假日，或当天只选 2–3 个点，避免堵车和过于疲劳。

| 时段 | 安排 |
|------|------|
| **早上** | **大鹏所城**（岭南古城）→ **较场尾**（「深圳小鼓浪屿」、海边民宿＋沙滩） |
| **中午** | 较场尾或鹿嘴山庄一带简餐/民宿餐厅 |
| **下午** | **桔钓沙**（玻璃果冻海）→ **杨梅坑**（礁石＋沿海骑行）→ **鹿嘴山庄**（《美人鱼》取景地、悬崖洞） |
| **傍晚** | 体力允许可去 **西涌/天文台** 看日落（路程较远，可酌情舍弃） |

## 🎫 四、门票预约与避坑指南

### 需提前预约/购票

| 项目 | 说明 |
|------|------|
| **大鹏半岛自驾** | 周末及节假日前住大鹏，自驾需在公众号预约通行，否则无法进入且易堵车 |
| **中英街** | 需提前预约并携带身份证，名额有限 |
| **深圳湾文化广场** | 已改为需购票入场，部分展馆收费；预约与参观顺序见专门笔记 |

### 避坑要点（来自多篇高赞笔记）

1. **多数场馆需公众号提前预约**（免费但限流），没约好容易白跑。
2. **周末海边人非常多**，大梅沙、桔钓沙一带堵车＋停车难，尽量工作日或错峰。
3. **深圳很多餐厅按人头收茶位费**（约 2–8 元/人），点单前可先问清。
4. **早晚高峰 1/3/5 号线很挤**，拖大件行李尽量避开。
5. **深圳紫外线强、公园/海边蚊虫多**，务必带好防晒和驱蚊液。
